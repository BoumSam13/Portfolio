WuXi AppTec - Starlims
Built with JavaScript, SQL, Starlims Scripting Language (SSL)
Starlims is a Laboratory Information Management System that was created by the company Abbott. At WuXi AppTec, we receive what are called enhancement requests to change or add different features to different areas of the system. These features help different departments complete their job duties.
Life Cycle Panel
The Life Cycle panel has different options that open up different applications. Most of the work that I did in Starlims was in the ACRS application. There is an Edit ACRS and a View ACRS application. Like most applications, depending on the role that the user has determines what they can see or have access to. For example some buttons may be enabled or disabled depending on the role. When the user first logs in he picks the role that he wants to log in with from a dropdown. The user will only be able to see the roles he is assigned to.
Edit ACRS
When Edit ACRS is clicked in the Life Cycle panel in the image above, then an application called Edit ACRS is opened in a new tab. I worked on the layout of the Edit ACRS application which in Starlims there is a toolbox, which has tables, text boxes, labels, etc. Also I used JavaScript, SQL, and the Starlims Scripting Language (SSL) for different types of functionality.
Edit ACRS Content
When an integer is typed into one of the text boxes and Enter is pressed or the link button is clicked, then data is shown in the two tables and in the three tabs. Each image shows the content in each panel. I worked on the layout of all three of the tabs and also added some new functionality.

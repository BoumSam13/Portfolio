WuXi AppTec - WuXi Scheduler
Built with PHP, Bootstrap, HTML4/5, mySQL
The application is built from an existing PHP scheduler. It was used to be able to use some existing functionality and add new functionality. It is used internally by different departments at multiple locations who need to schedule specific tasks. Before WuXi Scheduler was put in place, employees were using the Microsoft Outlook calendar to schedule tasks.
Main Page
This page is what is first shown when a user logs into the application. If the user has any Assays or Tasks assigned to them, they will be shown on this page. An Assay is like a project and the Tasks are the tasks in a project (Assay).
Browse Tasks Page
The page shows all of the tasks for all users. If you click the Browse tab, there is also a page for Browse Assays, which shows all of the assays for all users. On both pages, the selection can be filtered by choosing an option in a dropdown or typing into a textbox for the specific field. The Export button exports all rows shown into an Excel document. Multiple records can be updated all at one time by clicking the check box next to each record that will be updated and then click (*find out what button*). This will bring up a window that will allow the user to update multiple fields.
Sprints Page
On the Sprints page, it shows all of the Assays on the left and to the right of the assay are the tasks for the assay. The columns above the tasks are the status of the task. A task can be clicked and dragged into the column of the status it should have. Each task and assay can be clicked and a new window will open up with different fields that can be updated such as the user who should be assigned to the task. The Tasks Today tab also opens up a page that looks like this. The only difference is it shows all of the tasks that are due the day of looking at the page.

WuXi AppTec - Viral Clearance Tracking
Built with C#, JavaScript, jQuery, HTML, CSS, SQL, ASP.NET
This is one of the first projects I worked on when I started WuXi AppTec in 2014. It is an application that is used internally by employees in the Virology department.
Main Page
The page allows users with access to see all records by clicking the Search button or filtering down the selection by entering in specific text in one or more of the text boxes above. When the Export button is clicked, it exports the data shown below in an Excel document. If the user has more than view rights, then they can create a new record by clicking the Create New button. The Edit link next to each record is also available to edit information related to the record.
Create New Page
When the Create New button is clicked on the Main page, this is the page that displays. It allows the user to create a new record by specifying different information. Once the Save button is clicked, new tabs are created next to the Overview tab and they contain more fields to enter in specific information.
Edit Page
When the Edit link next to a record on the Main page is clicked, this is the page that is displayed. It allows the user to edit existing data for the specific record. If the user only has view rights, the Edit link will be shown as View. When clicked, this same page will be shown; however, none of the fields will be editable.

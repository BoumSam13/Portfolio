Bloc Jams is a project I created in the Full Stack Web Developer program at Bloc. The application was created with JavaScript and jQuery. Another version of the project was created and it uses AngularJS. This is an application that displays different albums and when an album is clicked, a new page is opened that shows information about the album and all of the songs, which can be played.
Main Page
This is the page that is shown when you first go to the application.
Collection
When the Collection link, shown in the upper left corner, is clicked; this is the page that is shown. It shows a list of albums. Currently, it is the same album, but the way it is set up is to show that this page is used to show all albums a user has accesss to.
Album
When an album is clicked on the Collection page, this page is shown. It shows the name of the album, the artist, when the album was released, and all songs on the album.
Songs
In order to play a song, hover the mouse over the number of the song and a play icon is displayed. When the play icon is clicked, the song starts playing, and the icon changes to a pause icon. To pause the song, the pause icon can be clicked or the pause in the controls in the bottom bar can be clicked. The user can also change what part of the song they are listening to by hovering the mouse over the small circle in the time bar and drag it to the time of the song they want to listen to. The volume can be adjusted the same way as well.

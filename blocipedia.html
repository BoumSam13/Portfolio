Bloc - Blocipedia
Built with Ruby, Rails, HTML, CSS, Bootstrap
Access the Code on GitHub
Blocipedia is a project I created in the Full Stack Web Developer program at Bloc. It is an application that allows users to create their own wikis. The user can upgrade to a premium member, which allows them to create private wikis and add/remove collborators to the private wikis.
Main Page
This is the page that is shown when you first go to the application.
Sign Up
This is the Sign Up page and once the Sign Up button is clicked, a confirmation email is sent to the email address that was entered in the Email field.

Log In
Once the link is clicked for the confirmation email, then a new browser window opens showing the following below.
Wikis
The Wikis page shows the users' wikis and all public wikis. If a user has the premium membership and a private wiki was created, other users would not be able to see that wiki unless they were given rights as a collaborator of the wiki.
New Wiki
This page is shown when the New Wiki button is clicked on the Wikis page.
Edit Wiki
After a new wiki is created, this page displays. It allows the user to edit the wiki by having the option to change the title or the body of the wiki. The wiki also can be deleted by clicking the Delete Wiki button.
User Account
If a user wants to update his name, email, or password; he should click the Edit Account tab. This page also gives the user the option to cancel their account.
Membership
In order for the user to see what membership he has or to update his membership, he should click the Membership tab.
Upgrade Membership
This page is shown if the Upgrade button on the page shown above is clicked. If the user decides to upgrade his account, he should click the Pay with Card button.
Enter Card Information
Once the Pay with Card button on the page shown above is clicked, this window is displayed for the user to enter in their email and card information.
